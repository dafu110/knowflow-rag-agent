# 销售合同审批

销售合同在提交审批前必须准备客户主体信息、合同正文、报价单、付款条款、交付计划和风险说明。

标准合同金额不超过 50 万元时，由销售经理和法务专员审批。超过 50 万元或包含非标准条款时，需要业务负责人、财务和法务负责人共同审批。

# 常见退回原因

合同被退回通常是因为客户名称与营业执照不一致、报价单版本错误、付款账期超过公司政策、缺少交付验收标准，或没有说明特殊折扣原因。

销售提交前应检查所有附件的版本号，并在审批备注中写明客户背景、预计签约时间和关键风险。
